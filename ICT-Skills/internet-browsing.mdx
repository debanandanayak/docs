1. The World Wide Web (WWW), or the Web, is defined as a huge collection of information.
2. It functions as a vast network of linked hypertext files stored globally on computers, serving as a platform for users to use or share information.

## Components of the WWW:

The WWW is composed of:
1. Web pag
2. Web browser
3. A system to transfer information between the web browser and the web pages

### Web Pages:

1. A web page is a document residing on a computer connected to the Internet.
2. Each web page possesses a unique address or location, similar to a house address. An example address is http://ncert.nic.in.
3. To view a web page, a web browser is required on the computer.

## Web Browsers:

1. A web browser is a software application essential for using and sharing information on the World Wide Web.
2. Popular examples include Google Chrome, Mozilla Firefox, Internet Explorer, Apple Safari, and Opera.
3. You can access information by typing the address in the browser’s address bar.


## Key Browser Features:

1. **Hyperlinks**: These are highlighted and underlined words on web pages. Clicking a hyperlink navigates you to another section on the same page or to a completely different page.
2. **Address bar (Alt+d)**: This box is where you type the address/URL of a web page and press Enter to display the corresponding page.
3. **Tabbed browsing (Ctrl+T)**: This feature allows a user to view multiple pages simultaneously within the same browser window.
4. **Back and forward buttons (Alt+Left or right arrow keys):** These buttons enable navigation back to the last page viewed or forward to the next page.
5. **Refresh button (F5)**: Used to reload a web page if it does not display completely or correctly.
### Steps for Searching Information

1. Open a web browser, such as Internet Explorer or Google Chrome.
2. Type http://google.co.in in the address bar.
3. Type your desired topic into the Google search bar (e.g., ‘best hill station in India’).
4. Press Enter.
5. Scroll and click on the desired link from the displayed search results.
